## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/yisleyen/go-exercises
```

Proje dizinine gidin

```bash
  cd go-exercises
```

İlgili klasördeki örneği çalıştırın

```bash
  go run main.go
```
